Quiz Game Idea

If time allows, incorporate the Math library into a new quiz game. The game will challenge users to memorize 10 digits of Pi, as well as the constant e, the square root of 2, and the square root of 1/2. These values can be accessed using the Math library:

  • Pi: Math.PI
  • e: Math.E
  • Square root of 2: Math.SQRT2
  • Square root of 1/2: Math.SQRT1_2

Additional Math Functions

Consider incorporating other Math functions or properties, such as:

  • Random number generation with Math.random() for quiz variations.
  • Rounding functions like Math.round() or Math.floor().
  • Trigonometric functions for angle-based questions.

Game Implementation

Utilize prompt to ask users questions about the values they need to memorize. At the end of the game, output the game statistics, such as the number of correct answers, total attempts, and a score based on their performance.
